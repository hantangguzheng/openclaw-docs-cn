什么是 OpenClaw？

私有化部署的个人 AI 助手平台，在你的设备上运行，不依赖任何云服务。

🖥️

本地优先架构

所有对话、配置、记忆都存储在你的本地设备上。数据永不离开你的控制，支持 Anthropic、OpenAI 或本地模型。

💬

全平台消息支持

通过你最熟悉的聊天工具与 AI 对话：WhatsApp、Telegram、Discord、Slack、Signal、iMessage、Microsoft Teams 等。

🧠

长期记忆

AI 会记住你的偏好和习惯，越用越懂你。记忆以 Markdown 文件存储，可编辑、可搜索、与 Obsidian 同步。

🔌

可扩展技能平台

从 ClawHub 安装 5000+ 社区技能，或让 AI 自己编写新技能。支持浏览器自动化、画布交互、语音对话等。

快速安装指南

一行命令完成安装，自动处理所有依赖

macOS / Linux

curl -fsSL https://openclaw.ai/install.sh | bash openclaw onboard

Windows (WSL2)

iwr -useb https://openclaw.ai/install.ps1 | iex openclaw onboard

已有 Node.js？

npm i -g openclaw # 或：pnpm add -g openclaw openclaw onboard
系统要求：Node.js 22.12.0 或更高版本。安装脚本会自动处理依赖，推荐使用一键安装方式。
