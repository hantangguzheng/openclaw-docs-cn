🔌 API 接口文档

OpenClaw 提供完整的 REST API，支持视频生成、图片生成、音乐生成等能力

快速开始

OpenClaw API 基于 HTTP REST 协议，认证方式为 API Key。

# 安装 OpenClaw CLI npm i -g openclaw # 配置 API Key openclaw config set api-key YOUR_API_KEY # 测试连接 openclaw api ping
Base URL：所有 API 请求均发送至 https://api.openclaw.ai/v1

认证

在请求头中携带 API Key：

Authorization: Bearer YOUR_API_KEY Content-Type: application/json

🎬 视频生成

POST /v1/video/generate

根据文本描述生成视频

参数类型必填说明
promptstring是英文视频描述 Prompt
durationnumber否视频时长（秒），默认 5，最大 60
aspect_ratiostring否宽高比：16:9 / 9:16 / 1:1，默认 16:9
stylestring否风格：cinematic / anime / realistic，默认 cinematic
modelstring否模型：openclaw-video / minimax-v3，默认 openclaw-video
# 请求示例 curl -X POST https://api.openclaw.ai/v1/video/generate \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "prompt": "a professional basketball player performing a slam dunk, dramatic lighting, cinematic", "duration": 5, "aspect_ratio": "16:9", "style": "cinematic" }' # 响应示例 { "id": "vid_abc123", "status": "processing", "estimated_time": 30, "urls": null }

🖼️ 图片生成

POST /v1/image/generate

根据文本描述生成图片

参数类型必填说明
promptstring是英文图片描述 Prompt
sizestring否尺寸：1024x1024 / 1792x1024 / 1024x1792，默认 1024x1024
nnumber否生成数量，默认 1，最大 4
modelstring否模型：dalle-3 / seedream，默认 dalle-3

🎵 音乐生成

POST /v1/audio/generate

根据描述生成音乐

参数类型必填说明
promptstring是音乐描述 Prompt
durationnumber否时长（秒），默认 30，最大 180
formatstring否格式：mp3 / wav，默认 mp3

📋 任务状态查询

GET /v1/tasks/{task_id}

查询异步任务的处理状态和结果

# 查询视频生成状态 curl https://api.openclaw.ai/v1/tasks/vid_abc123 \ -H "Authorization: Bearer YOUR_API_KEY" # 响应 { "id": "vid_abc123", "status": "completed", "urls": { "video": "https://cdn.openclaw.ai/videos/abc123.mp4", "thumbnail": "https://cdn.openclaw.ai/thumbnails/abc123.jpg" }, "metadata": { "duration": 5, "model": "openclaw-video" } }
