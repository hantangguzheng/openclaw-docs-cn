功能特性

OpenClaw 的完整功能一览——本地优先、多渠道、可扩展。

📡 集成一览

OpenClaw 连接 50+ 渠道、技能和模型提供商。按类别浏览：

  • 渠道 — WhatsApp、Telegram、Discord、Slack、Signal、iMessage、Teams、WebChat、Matrix、Zalo 等
  • 技能 — ClawHub 注册表（5000+ 技能）、生产力、开发、媒体、渠道类；使用 npx clawdhub@latest install 安装
  • 模型提供商 — Anthropic、OpenAI、Ollama、Gemini 及 40+ 其他；支持分级路由和故障转移

⚙️ 核心平台

Gateway 网关

WebSocket 管理层，提供会话管理、在线状态跟踪、配置控制、定时任务、Webhook 支持、控制界面、Canvas 托管，以及带有 token 使用量仪表板的 Web UI。

命令行界面

网关管理、Agent 控制、消息发送、安装向导和诊断命令。

Agent 运行时

以 RPC 模式运行的 Agent 运行时环境，支持工具流式输出和分块流式输出。

会话架构

主会话（直接对话）、隔离的群组会话、可配置的激活模式、队列管理和回复处理。

主动运行

Agent 可以按计划运行（心跳、定时任务），无需你发起对话即可"醒来"执行任务、简报或检查。参见自动化部分。支持非阻塞模式——你发送请求后可以继续工作，Agent 在后台运行。

媒体处理

图像、音频和视频处理，带转录集成、文件大小限制和临时文件管理。

🛠️ 工具与自动化

浏览器自动化

独立的 Chrome/Chromium 浏览器实例，提供全面的自动化功能：

  • 自动网页导航和页面交互
  • 表单填写和自动提交
  • 网站数据提取和解析
  • 屏幕截图和快照生成
  • 自动化文件上传
  • 浏览器配置文件配置和管理

交互式画布 (Canvas)

Agent 使用 A2UI 框架控制的动态可视化工作空间：

  • A2UI 界面推送和重置操作
  • 浏览器内 JavaScript 代码执行
  • 画布状态捕获和快照功能
  • Agent 操作的视觉交互环境

定时任务与自动化

  • Cron 任务 — 安排重复性任务
  • 唤醒 — Agent 定时唤醒执行任务
  • Webhook — 通过 HTTP 端点的外部触发器
  • Gmail Pub/Sub — 实时邮件触发

技能平台

  • 内置、管理和工作区技能
  • 安装门控和 UI
  • 社区技能注册表（ClawHub）
  • 自我改进：可编写自己的技能
实际使用中：我用浏览器自动化来做研究和数据抓取；用定时任务做每日简报和多 Agent 心跳设置。技能是大部分"魔法"所在——浏览 ClawHub 找社区技能，或让 OpenClaw 为你写一个。

📱 应用与节点

macOS 应用

  • 菜单栏控制面板
